## Getting Started

To get a local copy up and running follow these simple example steps.

1. `git clone https://github.com/macnick/Weather-App.git`
2. `cd Weather-App`
4. `npm i`
6. Use live server on VSCode to open the index.html with your favorite browser

### Prerequisites

- A modern browser
